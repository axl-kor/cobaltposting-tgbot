# Cobalt Posting Bot
This bot can be useful for those who use @coboltbot for downloading content and needs custom formatting for posts.

This bot offers you:
 - Custom formatting for downloaded posts
 - Some statistics about your downloads (even size of all files)
 - Friendly setup and settings

## How do i use it?
1. You set up bot for posting to your channel
2. You send this bot a link to content
3. Bot downloads it, applies custom formatting(if you setted it) to the caption, and sends this message to your channel.

**config.json schema**
```json
{
    "token": "",
    "nodes": ["https://..."],
    "errors_report_chatid": 123,
    "admins": [123, 456]
}
```
